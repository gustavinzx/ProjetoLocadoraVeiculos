# 🚗 Locadora de Veículos - Gestão de Frota

Este repositório contém o código-fonte e a documentação do **Trabalho Final da disciplina de Teste de Software (2026)**. O sistema consiste em uma aplicação Java para gerenciamento de locadoras, integrando banco de dados SQLite, API REST e uma interface CLI.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java 17
* **Build System:** Maven
* **Banco de Dados:** SQLite
* **Testes:** JUnit 5, Postman & Newman
* **Infraestrutura:** GitLab CI/CD

---

## 🚀 Pipeline de Testes e CI/CD
Para este projeto, optamos por utilizar o **GitLab** para a execução da pipeline de CI/CD. Essa escolha técnica permitiu o uso de ferramentas nativas de visualização de estágios, integração contínua e relatórios automáticos de cobertura e sucesso dos testes.
