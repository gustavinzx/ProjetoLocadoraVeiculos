# 🚗 Locadora de Veículos - Gestão de Frota

Este repositório contém o código-fonte e a documentação do **Trabalho Final da disciplina de Teste de Software (2026)**. O sistema consiste em uma aplicação Java para gerenciamento de locadoras, integrando banco de dados SQLite, API REST e uma interface CLI.[cite: 3, 4]

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java 17[cite: 3, 4]
* **Build System:** Maven[cite: 3, 4]
* **Banco de Dados:** SQLite[cite: 3, 4]
* **Testes:** JUnit 5, Postman & Newman[cite: 3, 4]
* **Infraestrutura:** GitLab CI/CD[cite: 3, 4]
